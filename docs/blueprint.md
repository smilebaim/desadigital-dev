# **App Name**: Remau Bako Tuo - Dev

## Core Features:

- Welcome Section: Display a welcome message with the village name.
- Village Info: Showcase village description.
- Bottom Navigation: Provide a navigation menu at the bottom of the page, offering access to key sections.

## Style Guidelines:

- Primary color: A muted yellowish-green (#D4E157), suggestive of new growth and rice paddies, the village economy's focus.
- Background color: A very light yellowish-green, almost white (#F9FCE6). This provides a clean and airy feel, contrasting with the primary color.
- Accent color: A desaturated orange (#D2691E) adds a rustic, natural warmth, that makes navigation elements more accessible.
- Font: 'PT Sans' (sans-serif) for both headlines and body text. A legible and modern font, suitable for the descriptive content about the village.
- Use simple, outlined icons for the bottom navigation bar. Ensure icons are easily recognizable.
- The page layout features a hero section with a descriptive block of text. Use clear divisions with sufficient white space. Bottom navigation is a must.
